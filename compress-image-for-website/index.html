Frequently Asked Questions

网站图片应该多大？

Google建议单张图片控制在200KB以内。Hero图片可以稍大，但不要超过500KB。

WebP格式有什么优势？

WebP格式比JPG小25-35%，比PNG小25%，同时保持相同质量。

图片大小会影响SEO吗？

是的。图片过大会导致页面加载慢，影响Google排名。Google Core Web Vitals会评估图片加载性能。

怎么批量压缩网站图片？

使用我们的工具逐个压缩，或者使用WordPress插件自动压缩。
